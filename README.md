# RULEX UI

该项目是全新的RULEX边缘网关框架专属UI，主要用来给开发者提供统一便利的前端体验。

## 环境搭建

Install `node_modules`:

```bash
npm install
```

or

```bash
yarn
```

### Start project

```bash
npm start
```

### Build project

```bash
npm run build
```

### Check code style

```bash
npm run lint
```

You can also use script to auto fix some lint error:

```bash
npm run lint:fix
```

### Test code

```bash
npm test
```
